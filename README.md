## Beüzemelés

### Backend:

cd backend

npm install: Függőségek telepítése.

node server.js: Szerver elindítása.


### Frontend:

cd frontend

npm install: Függőségek telepítése.

ng serve: Szerver elindítása.


### Weboldal elérése

A weboldal elérhető böngészőből a következő címen:

http://localhost:4200
